# COMPASS

AI research discovery platform — search, rank, and track papers from arxiv with a terminal-inspired interface.

**Live:** [https://alvi-uiu.github.io/COMPASS/](https://alvi-uiu.github.io/COMPASS/)

## Features

- **Feed** — fetch latest arxiv papers filtered by AI subtopics (AI, ML, NLP, CV, Robotics, Security, etc.)
- **Search** — full-text search across OpenAlex with BM25 + citation + recency combined ranking
- **Trending** — compute trend scores from recency, citations, author count, and AI relevance
- **Learn** — built-in educational content on AI/ML concepts
- **Collections** — create, rename, and export named paper collections
- **News** — AI-related stories from Hacker News
- **Saved** — bookmark papers and export as text
- **Settings** — configure papers per fetch, lookback days, abstract preview length, and default sort

Built with vanilla JS — no build tools, no dependencies.
